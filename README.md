# SeeSomethingSaySomething

Simple React prototype

## Stack

- React + Vite
- React Router
- CSS

## Pages

- Home (`/`)
- Send Request (`/send-request`)
- Galery (`/galery`)

## Run Locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start development server:

   ```bash
   npm run dev
   ```

3. Open the URL displayed by Vite (usually `http://localhost:5173`).
